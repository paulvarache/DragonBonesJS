## How to build
* Clone or download [DragonBonesJS](https://github.com/DragonBones/DragonBonesJS/).
* Install [Node.JS](https://nodejs.org/).
* Open `DragonBonesJS/Hilo/1.x/` in command.
* $ `npm install`
* $ `npm run build`

## Make sure project structure like this:
```
Your project
    |-- libs
        |-- hilo.d.ts
    |-- node_modules
        |-- ...
    |-- out
        |-- ...
    |-- src
        |-- ...
    |-- ...
```
