## How to run

## Make sure project structure like this:
```
Your project
    |-- assets
        |-- Scene
            |-- ...
        |-- Script
            |-- dragonBones
                |-- dragonBones.js
            |-- ...
        |-- resources
            |-- dragonBones files
            |-- ...
    |-- declaration
        |-- dragonBones.d.ts
        |-- ...
    |-- creator.d.ts
    |-- tsconfig.json
    |-- ...
```

## Notice
You should remove all dragoneBones declaration in creator.d.ts.
